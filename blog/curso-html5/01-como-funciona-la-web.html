01. ¿Cómo funciona la Web?

Curso HTML5 desde cero — Freimel Jerez

Antes de aprender a crear páginas web, es importante entender cómo funciona la web. Todo comienza cuando una persona utiliza un computador o un celular conectado a Internet.

👤 El usuario

El usuario es la persona que navega por Internet usando un dispositivo como un computador, un celular o una tablet.

🌐 El navegador

El navegador es el programa que el usuario utiliza para acceder a Internet, como Chrome, Edge o Firefox.

🔗 La dirección web (URL)

El usuario escribe una dirección web en el navegador. Esa dirección indica a qué sitio quiere acceder.

🌍 Internet

Internet es una red mundial que conecta millones de computadores en diferentes partes del mundo.

📡 DNS

El sistema DNS se encarga de traducir el nombre del sitio web a una dirección que los computadores puedan entender.

🖥️ El servidor

El servidor es un computador que almacena la información del sitio web y está siempre conectado a Internet.

📤 Petición

El navegador envía una petición al servidor solicitando la información del sitio web.

📥 Respuesta

El servidor responde enviando la información solicitada al navegador.

🧠 Interpretación

El navegador recibe la información y la procesa para poder mostrarla correctamente.

🖼️ Página web

Finalmente, el usuario ve la página web en pantalla. Todo este proceso ocurre en cuestión de segundos.

✅ Resumen

02 - Selectores Básicos

En esta clase aprenderás cómo seleccionar elementos HTML usando CSS.

📌 ¿Qué es un selector?

Un selector en CSS permite indicar qué elemento HTML queremos modificar con estilos.

🔹 Selector por etiqueta

p {
  color: blue;
}

Este selector aplica el estilo a todos los elementos <p> del documento.

🔹 Selector por clase

.destacado {
  color: red;
}
<p class="destacado">Texto importante</p>

🔹 Selector por ID

#titulo {
  font-size: 30px;
}
<h1 id="titulo">Hola Mundo</h1>
